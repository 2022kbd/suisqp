How to Use

Placing Components

  • Select a component from the palette on the left
  • Click anywhere on the workspace to place it
  • Press Escape or click Select to switch back

Connecting Wires

  • Use the Select tool
  • Click an output port (right side) to start a wire
  • Click an input port (left side) to complete it
  • Press Escape to cancel a wire

Interacting

  • Toggle switches by clicking the switch toggle
  • Drag components to rearrange
  • Select a component or wire, then press Delete or the trash button

Logic Gates

  • AND — Output is ON only when ALL inputs are ON
  • OR — Output is ON when ANY input is ON
  • NOT — Inverts the input (ON → OFF, OFF → ON)
  • XOR — Output is ON when inputs DIFFER

Tip: Try the Speaker output for a surprise!
